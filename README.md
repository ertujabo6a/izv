# Data Analysis and Vizualization Project 2023
The goal of the project is to learn how to collect, process, and analyze data available on the internet.

## Project Structure Overview
### Part 1
The first part of the project will focus on verifying adoption of techniques for efficient data processing, visualization, and gathering.

- Efficient numerical calculations
- Simple visualization
- Data downloading and processing

### Part 2
- Various data perspectives
- Advanced visualization of results
- Conclusions processing (understanding the data)

### Overall Project
- Data representation on a map, operations on this data
- Correlation and prediction
- Automatic generation of report sections
- Compilation into an analytical report
